##Dropmark

This shows a random image from a dropmark.com collection every time a slide is displayed.

---

###Parameters

* **username**       (required)
* **collection_id**  (required)


###Example Url

`http://your-url.com/custom_slides/dropmark/?username=your_user_name&collection_id=123456`
